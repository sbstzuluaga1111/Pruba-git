git branch:
mira la rama actual en la que me encuetro.

git branch rama1:
crea un rama.

git branch -m rama1 prueba1:
renombra la rama o cambia el nombre.

git checkout prueba1:
selecciona la rama donde vas a trabajar.

git branch -d prueba1:
elimina una rama pero no puedes estar trabajando en ella.

git diff master prueba1:
Mirar cambios de una rama y otra.

git merge prueba1 master:
combina los cambios de las ramas hacia master pero hay que estar parado o trabajando en la rama a la que se le van a adjuntar esos cambios.
